This lab focuses on responsive web design. Using Bootstrap I created a responsive website that can be viewed from various devices.

Why Use Bootstrap?

  • Easy to Use: Requires a basic knowledge of HTML and CSS.
  • Responsive Design: Adapts to phones, tablets, and desktops.
  • Mobile-First: Designed with mobile devices as a priority.
  • Cross-Browser Support: Works smoothly on all modern browsers.
